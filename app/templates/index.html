Access Yale Dining data with ease!

YaleDine provides an open, standards-compliant API that developers can use to access Yale Dining's dining hall and menu data. It replaces Yale Dining's deprecated official API, and scrapes data from Yale Dining's occupancy statistics page, the new JAMIX menus website, and various other sources, compiling it together and cleaning it up for presentation in a polished and developer-friendly API that can easily be used for your projects.

For questions, comments, bug reports, or suggestions, contact erik.boesen@yale.edu.

API documentation

Endpoints

All endpoints below are from the root https://yaledine.com/api. For example, https://yaledine.com/api/halls. All endpoints return a JSON response.

/halls

Returns a list of available dining Hall objects.

/halls/:id

Returns a single Hall object with the given id.

/halls/:id/managers

Returns a list of Manager objects for the given hall.

/halls/:id/meals

Returns a list of Meal objects for the given hall.

/meals/:id

Returns a single Meal object with the given id.

/meals/:id/items

Returns a list of Item objects for the given meal.

/items/:id

Returns a single Item objects with the given id.

/items/:id/nutrition

Returns a single Nutrition object for the Item with the given id.
