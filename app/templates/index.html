Access Yale Dining data with ease!

YaleDine provides an open, standards-compliant API that developers can use to access Yale Dining's location and menu data. It replaces Yale Dining's deprecated official API, and scrapes data from Yale Dining's occupancy statistics page, the new JAMIX menus website, and various other sources, compiling it together and cleaning it up for presentation in a polished and developer-friendly API that can easily be used for your projects.

For questions, comments, bug reports, or suggestions, contact erik.boesen@yale.edu.

API documentation

Endpoints

All endpoints below are from the root https://yaledine.com/api. For example, https://yaledine.com/api/locations. All endpoints return a JSON response.

/locations

Returns a list of available dining Location objects.

/locations/:id

Returns a single Location object with the given id.

/locations/:id/managers

Returns a list of Manager objects for the given location.

/locations/:id/meals

Returns a list of Meal objects for the given location.

/meals/:id

Returns a single Meal object with the given id.

/meals/:id/items

Returns a list of Item objects for the given meal.

/meals/:id/courses

Returns a list of Course objects for the given meal.

/courses/:id

Returns a single Course object with the given id.

/courses/:id/items

Returns a list of Item objects for the given course.

/items/:id

Returns a single Item objects with the given id.

/items/:id/nutrition

Returns a single Nutrition object for the Item with the given id.
